# LearnMusic

一个用于练习音乐听感的小程序项目。第一版目标是做 C 大调数字谱听音训练：先用方向热身，再用音程对比和音高入门建立听感，最后做综合挑战。

## 当前设计方向

- 先做本地浏览器版本，打开网页即可练习。
- 第一版只使用 C 大调音阶，也就是数字谱 1-7，不依赖服务器。
- 四种训练都可以随时进入，不做强制闯关。
- 音程先用二选一对比训练，不直接七选一考试。
- 音高拆成多个阶梯卡片：稳定音、邻近音、上方邻近、倾向音、全部音。
- 答题后显示正确答案，音高用 1-7 显示，不用 C/D/E/F/G/A/B。
- 答对播放开心提示音，答错播放低落提示音，并显示积分、连击和徽章。
- 使用 Web Audio API 直接生成声音。
- 练习结果保存在本地浏览器中。

## 本地运行

```powershell
.\scripts\serve.ps1
```

默认发布到 `http://127.0.0.1:8200/`。
